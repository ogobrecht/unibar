This example shows how a unicode bar chart is looking - with or without filling the scale.

  • p_scale: 100
  • p_width_block_characters: 25
  • There are 8 different block characters available: 1/8, 1/4, 3/8, 1/2, 5/8, 3/4, 7/8 and 1
  • The value of one block character is then 4 (100 / 25) - you can see this easily on the missing white space after a non full block when the scale is filled with shade
  • The maximum precision is then 0.5 (4 / 8)
  • We use the Font 'Arial Unicode MS' which have all the needed unicode block characters available and should be normally available on most Windows and Mac OS X systems
